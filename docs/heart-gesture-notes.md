# Heart Gesture Notes

爱心手势的识别思路主要来自双手关键点之间的空间关系。

## 核心特征

1. 双手协作：必须检测到两只手，且两只手在画面中大致对称分布。
2. 拇指特征：双手拇指指尖，也就是 MediaPipe Hands 的关键点 `4`，相互贴近或触碰，形成爱心下尖角。
3. 食指特征：双手食指指尖，也就是关键点 `8`，相互贴近或触碰；食指第二关节 `7`、第三关节 `6` 与指尖 `8` 形成约 120 到 150 度的弯曲角，构成爱心上侧轮廓。
4. 其他手指：中指、无名指、小指，也就是关键点 `12`、`16`、`20`，自然弯曲贴合手掌，不应大幅伸展。
5. 空间关系：双手拇指和食指围成接近闭合的心形轮廓。
